# Conversor de Unidades de Comprimento

Este projeto é uma ferramenta web simples e eficiente para a conversão de diferentes unidades de medida de comprimento (Metros, Quilômetros, Milhas e Polegadas), desenvolvido de forma 100% frontend para a disciplina de Desenvolvimento Colaborativo.

## Funcionalidades Principais

* Conversão instantânea entre múltiplos seletores de comprimento;
* Tratamento de erros e validações para impedir cálculos com valores vazios ou negativos;
* Interface limpa, intuitiva e direta ao ponto;
* Elementos encapsulados em um container estilo card para melhor legibilidade;
* Código estruturado e componentizado de forma colaborativa via Git.

## Tecnologias Utilizadas

* **HTML5:** Estrutura semântica dos blocos de entrada e saída.
* **CSS3:** Estilização moderna e layout responsivo.
* **JavaScript (ES6):** Lógica matemática de conversão e manipulação do DOM.
* **Git & GitHub:** Versionamento de código e gerenciamento de branches.
